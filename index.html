Explore a handmade fairytale kingdom, speak with its citizens, solve object puzzles, and follow the trail from the castle yard to the Chapel of Noon.

System requirements

Mac

Apple Silicon Mac recommended. Download includes the game and required runtime files.

Windows

Windows 10 or 11, 64-bit. An NVIDIA RTX GPU is recommended for fast local character conversations.

Optional dialogue model

For enhanced local NPC conversations, place Qwen3.5-4B-Q4_K_M.gguf in refs/models. The game still runs without it.
